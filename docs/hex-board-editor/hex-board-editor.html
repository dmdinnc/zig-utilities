How to Use:

  • Colors Tab: Paint hex cells and add symbols (circle, crosshair, person)
  • Lines Tab: Draw lines and arrows between hex centers
  • Left-click + drag: Paint cells or draw lines between hexes
  • Right-click: Add colored symbols to hex cells (Colors tab) or remove lines (Lines tab)
  • Symbol Selection: Choose symbol type before right-clicking on hexes
